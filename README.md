Project.1
숫자야구게임
18.6.19 - 18.6.23

임의로 선택된 세자리 숫자를, 입력을 통해 추측해서 맞추는 게임

- 사용되는 숫자는 0에서 9까지 서로 다른 숫자이다.

- 숫자는 맞지만 위치가 틀렸을 때는 볼.

- 숫자와 위치가 전부 맞으면 스트라이크.

- 숫자와 위치가 전부 틀리면 아웃.
